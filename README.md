4. Опис на проектот

Miginator е игра базирана на играта Akinator, која ги погодува учениците од класот III-1 и професорите од МИГ во зависност од тоа како ќе му се одговори на прашањата.

Што прави проектот: Играчот си замислува една личност, Miginator му поставува прашања и во зависност од одговорите кои ќе ги добива, пробува да ја погоди личноста.

За кого е наменет: Наменет е за сите професори и ученици од класот III-1.

Зошто е корисен: Miginator забавна игра за учениците, а за професорите може да служи како интересен начин да ги запамтат учениците од класот.
